About MailPro Nafij

A modern, secure Gmail management platform

🏠 Home 🚀 Get Started

🎯 Our Mission

MailPro Nafij provides a secure, streamlined way to access and manage multiple Gmail accounts from a single platform. Built with modern web technologies and Google's OAuth2 authentication, we ensure your email data remains safe while giving you powerful management tools.

✨ What Makes Us Different

🔐 OAuth2 Security

Your Gmail password never touches our servers. We use Google's official OAuth2 protocol for authentication, ensuring maximum security.

👥 Multi-Account

Connect and switch between multiple Gmail accounts effortlessly. View, manage, and organize all your email from one dashboard.

✉️ Email Composition

Send emails directly with support for HTML formatting, multiple recipients (To, Cc, Bcc), and real-time preview.

👁️ Multi-View Email

View emails in plain text, rendered HTML, or raw source format. Switch between views instantly for any email.

⚙️ Admin Dashboard

Powerful admin controls with account locking, quota management, audit logs, sending statistics, and bulk operations.

⚡ Auto-Refresh

Inbox automatically refreshes every 30 seconds so you never miss an important email. Manual refresh also available.

🛠️ Built With

Node.js & Express Backend API Server
MongoDB Atlas Cloud Database
Google APIs Gmail & OAuth2
HTML5 & CSS3 Modern Frontend
